# 3D 音乐圣诞树

## 项目简介
**3D 音乐圣诞树** 是一个基于 **Three.js** 的交互式 3D 音乐可视化网页项目。  
它将圣诞音乐的节奏、频谱与 3D 圣诞树灯光和粒子特效实时结合，打造沉浸式、冰雪风格的节日体验。

无需安装环境，**直接打开 HTML 文件即可运行**。

---

## 主要特色

- **音乐可视化**  
  使用实时音频频谱分析，将音乐节奏映射到圣诞树的灯光亮度、颜色和动画效果。

- **完全交互式 3D 场景**  
  - 沉浸式探索整棵 3D 圣诞树

- **随机圣诞祝福**  
  每次选择音乐，都会显示不同的圣诞祝福语，增加趣味性。

- **灵活音乐选择**  
  - 内置 5 首精选圣诞音乐  
  - 支持上传本地音乐文件进行可视化

---

## 快速体验

1. **选择音乐**
   - 点击任意一首内置圣诞歌曲  
   - 或上传你自己的音乐文件

2. **享受特效**
   - 观察音乐驱动的彩色灯光变化  
   - 感受节奏同步的粒子与光效动画

---

## 技术亮点

- **Three.js**  
  高性能 WebGL 3D 渲染引擎，构建圣诞树与场景

- **Web Audio API**  
  实时音频解码与频谱分析，实现音乐驱动视觉效果

- **实时着色器（Shader）特效**  
  灯光、泛光（Bloom）与粒子动画系统

- **响应式设计**  
  自适应不同屏幕尺寸，兼容桌面与移动设备
